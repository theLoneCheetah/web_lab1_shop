# Магазин

[Деплой на GitHub Pages](https://thelonecheetah.github.io/web_lab1_shop/)

## Задача
* Разработать интернет-магазин, где пользователь сможет просматривать каталог товаров, добавлять их в корзину и управлять покупками.
* Не использовать CSS-фреймворки (Bootstrap, Foundation и т.д.) и JS-фреймворков (Angular, React, Vue и т.д.).
* Предоставить deploy проекта.

## Реализованные функции
* HTML-страница магазина с карточками товаров, названием, ценой и кнопками "Добавить в корзину".
* Использованы инструменты CSS: flex/grid и семантическая разметка (header, main, section, article, aside, footer).
* Реализовано добавление товара, удаление товара из корзины, изменение количества (кнопки + и -), корректное отображение при любых условиях.
* Сайт корректно отображается на ноутбуке, планшете и телефоне.
* Форма открывается но кнопке "Оформить заказ" в модальном окне.
* Форма содержит поля: имя, фамилия, адрес доставки, номер телефона и кнопку "Создать заказ".
* После заполнения всех полей и нажатия "Создать заказ" появляется сообщение "Заказ создан!" и происходит очистка корзины.
* Реализовано сохранение в localStorage для подгрузки корзины при обновлении страницы.
