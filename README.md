# Магазин

## Задача
* Разработать интернет-магазин, где пользователь сможет просматривать каталог товаров, добавлять их в корзину и управлять покупками.
* Не использовать CSS-фреймворки (Bootstrap, Foundation и т.д.) и JS-фреймворков (Angular, React, Vue и т.д.).
* Предоставить deploy проекта.
